# Web-Admin Anthropic Credentials Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the template-user `.credentials.json` copy mechanism and the three Anthropic env-var config fields with a single admin-managed web panel that stores `ANTHROPIC_API_KEY` / `ANTHROPIC_BASE_URL` / `ANTHROPIC_AUTH_TOKEN` in the DB `settings` table and injects the non-empty ones into every user's terminal.

**Architecture:** Admin edits three fields on the panel → `PUT /api/admin/settings/anthropic` upserts three `settings` rows. At PTY spawn, `buildUserEnvFactory` reads those rows via `resolveAnthropic()` and passes a `pty.AnthropicCreds` struct to `BuildUserEnv`, which sets only the non-empty vars. The `.credentials.json` copy path, the template-user HTTP handlers/routes, the `CLAUDE_TEMPLATE_USER` env var, and the three Anthropic env-var config fields are all deleted.

**Tech Stack:** Go 1.26, chi, modernc.org/sqlite, plain JS SPA (built by Dockerfile, embedded via `go:embed`).

```go
package pty

import (
	"fmt"
	"os"
	"sort"
	"strings"

	"github.com/ldm0206/claude-docker/backend/internal/config"
)

const sharedClaudeBin = "/opt/claude/bin"

// AnthropicCreds carries the three admin-managed Anthropic values that are
// injected into a user terminal only when non-empty.
type AnthropicCreds struct {
	APIKey    string
	BaseURL   string
	AuthToken string
}

func BuildUserEnv(cfg *config.Config, creds AnthropicCreds, username, claudeConfigDir string) []string {
	envMap := make(map[string]string, 32)
	for _, e := range os.Environ() {
		if key, val, ok := strings.Cut(e, "="); ok {
			envMap[key] = val
		}
	}
	set := func(k, v string) {
		envMap[k] = v
	}
	set("HOME", fmt.Sprintf("/home/%s", username))
	path := envMap["PATH"]
	set("PATH", fmt.Sprintf("%s:%s", sharedClaudeBin, path))
	if _, ok := envMap["TERM"]; !ok {
		set("TERM", "xterm-256color")
	}
	if _, ok := envMap["COLORTERM"]; !ok {
		set("COLORTERM", "truecolor")
	}
	set("CLAUDE_CONFIG_DIR", claudeConfigDir)
	if creds.APIKey != "" {
		set("ANTHROPIC_API_KEY", creds.APIKey)
	}
	if creds.BaseURL != "" {
		set("ANTHROPIC_BASE_URL", creds.BaseURL)
	}
	if creds.AuthToken != "" {
		set("ANTHROPIC_AUTH_TOKEN", creds.AuthToken)
	}
	for _, p := range []struct{ hi, lo, val string }{
		{"HTTP_PROXY", "http_proxy", cfg.HTTPProxy},
		{"HTTPS_PROXY", "https_proxy", cfg.HTTPSProxy},
		{"ALL_PROXY", "all_proxy", cfg.AllProxy},
	} {
		if p.val != "" {
			set(p.hi, p.val)
			set(p.lo, p.val)
		}
	}
	set("NO_PROXY", cfg.NoProxy)
	set("no_proxy", cfg.NoProxy)
	set("API_TIMEOUT_MS", fmt.Sprintf("%d", cfg.APITimeoutMS))
	env := make([]string, 0, len(envMap))
	for k, v := range envMap {
		env = append(env, k+"="+v)
	}
	sort.Strings(env)
	return env
}
```

This deletes `BuildClaudeEnv` and `claudeBin` (only `BuildClaudeEnv` used them). `sharedClaudeBin` is retained.

```go
// resolveAnthropic reads the three admin-managed Anthropic settings. A missing

func (s *Server) resolveAnthropic() system.AnthropicCreds {
	get := func(k string) string {
		v, err := s.db.GetSetting(k)
		if err != nil {

		}
		return v
	}
	return system.AnthropicCreds{
		APIKey:    get(apiKeyKey),
		BaseURL:   get(baseURLKey),
		AuthToken: get(authTokenKey),
	}
}
```
